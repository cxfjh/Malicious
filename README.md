## 使用

1. 克隆项目目录：

```bash
git clone https://github.com/cxfjh/Malicious.git
```

2. 如果你没有 `local.properties` 文件，请创建一个并在其中指定你的 Android SDK 路径：

```properties
sdk.dir=/path/to/your/android/sdk
```

3. 使用 Android Studio 打开项目，或者通过命令行构建项目：

```bash
./gradlew build
```
